## 早まった一般化

- 説明
    - 十分な論拠がない状態で演繹的な一般化を行うこと
- 誤謬の例
    - 1, 2, 3, 4, 5, 6はいずれも120の約数だ。よってすべての整数は120の約数である。
    - 100匹のカラスを見た。全てのカラスは黒かった。よってカラスは黒い。
- 筆者的見解
    - 帰納法のバグ

## 前後関係と因果関係の混同（前後即因果の誤謬）

- 説明
    - A が起きてから B が起きたという事実を捉えて、A が B の原因であると早合点すること
- 誤謬の例
    - あなたがこのアパートに引っ越してくるまで暖房設備には何の問題もなかった
    - 高校や大学への進学率はかつてないほど高くなってきた。それでも、少年犯罪や問題を抱えた若者はかつてより多い。よって、若者が教育によって堕落させられている。
- 筆者的見解
    - 「因果関係」と「前後関係」と「相関関係」は違う
    - 前後関係はあるように見えるが、因果(原因と結果)は伴っていない

<br>

## 相関関係と因果関係の混同（擬似相関）

- 説明
    - 相関関係があるものを短絡的に因果関係があるものとして扱う
- 誤謬の例
    - 撲滅された病気の数とテレビの普及には相関関係がある。よってテレビが普及すれば病気が撲滅される。
    - アイスクリームの売り上げが最も高い時期には、プールでの溺死事故も最も多い。アイスクリームの売り上げ増が溺死増の原因。
- 筆者的見解
    - 「因果関係」と「前後関係」と「相関関係」は違う
    - 相関しているように見えるが、因果(原因と結果)は伴ってない

<br>

## 因果関係の逆転

- 説明
    - 因果関係を逆転させて主張する
- 誤謬の例
    - 火災現場に出動する消防士が多いほど火災の規模は大きい。従って、出動する消防士が多くなることが、火災が大きくなる原因だ。
    - 車椅子は危険である。なぜなら、車椅子に乗っている人は事故に遭ったことがあるから。
    - バスケットボールの選手は身長が高い。よってバスケットボールをすると背が伸びる。
- 筆者的見解
    - 「因果関係」と「前後関係」と「相関関係」は違う
    - 相関しているように見えるが、因果(原因と結果)が逆転している

<br>

## テキサスの狙撃兵の誤謬（クラスター錯覚）

- 説明
    - 本来相関のないものを相関があるとして扱う。
- 誤謬の例
    - 上官が狙撃兵に腕前を問うたところ、遠くにある壁の標的の真中に命中しているのを指し示したため腕前に感心したが、実際は壁の銃痕にあとから標的を描いただけだった。
    - 電力線が健康上の悪影響を引き起すかどうか調査したところ、近いところに住む人々の小児白血病の発生率は4倍だったが、実際は多重比較問題によって統計的に有意差が出ただけだった。
- 筆者的見解
    - 「因果関係」と「前後関係」と「相関関係」は違う
    - 相関しているように見えるが、そもそも相関関係などない

## [WIP]間違った類推

- 説明
    - 重大な相違を無視して事象の類似性に基づいて論証（類推）すること。
- 誤謬の例
    - 「酒とコーヒーは似たような嗜好品だ。飲酒は法律で規制されている。よってコーヒーを飲むのは法律で規制されているはずだ」。

<br>

## [WIP]例外の撲滅

- 説明
    - 例外を無視した一般化を元に論旨を展開すること。
- 誤謬の例
    - ナイフで人に傷をつけるのは犯罪だ。外科医はナイフで人に傷をつける。従って、外科医は犯罪者だ。

<br>

## [WIP]滑り坂論法

- 説明
    - 「風が吹けば桶屋が儲かる」的な論法で、何らかの事物の危険性を主張すること。
    - ドミノ理論。
    - 必ずしも誤謬とは限らない。
    - 「風が吹けば桶屋が儲かる」は誤謬といってもよいが、「第一次世界大戦でロシア軍が連戦連敗だとコーカサスバイソンが絶滅する」は現実に起こった事態である。

<br>

## [WIP]論点先取

- 説明
    - 結論を前提の一部として明示的または暗黙のうちに使った論証
    - 形式的には間違っていないが、結論が前提の一部となっているため、全体として真であるとは言えない。
- 誤謬の例
    - 彼は正直者なんだから、ウソを言うわけないじゃないか

## [WIP]曖昧語法

- 説明
    - 文法的に曖昧な文形で主張をすること。
- 誤謬の例
    - 十代の若者に自動車を運転させるべきではない。それを許すのは非常に危険だ
        - 若者が危険な目にあうと言っているのか、若者が他者を危険にさらすと言っているのか曖昧である。

## [WIP]連続性の虚偽

- 説明
    - 述語の曖昧性により常識的な認識とのズレが生じる誤謬
    - 「砂山のパラドックス」、「テセウスの船」とも。
- 誤謬の例
    - 「砂山から砂粒を一つ取り出しても、砂山のままである。さらにもう一粒取り出しても砂山である。したがって砂山からいくら砂粒を取り出しても砂山は砂山である」。

## [WIP]多重質問の誤謬

- 説明
    - 質問の前提に証明されていない事柄が含まれており、「はい」と答えても「いいえ」と答えてもその前提を認めたことになるという質問形式
- 誤謬の例
    - 「君はまだ天動説を信じてるのかね？」という質問は、「はい」でも「いいえ」でも「過去に天動説を信じていた」という暗黙の前提を認めたことになる。
