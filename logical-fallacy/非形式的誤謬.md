## 早まった一般化

- 説明
    - 十分な論拠がない状態で演繹的な一般化を行うこと
- 誤謬の例
    - 1, 2, 3, 4, 5, 6はいずれも120の約数だ。よってすべての整数は120の約数である。
    - 100匹のカラスを見た。全てのカラスは黒かった。よってカラスは黒い。
- 筆者的見解
    - 帰納法のバグ

- 誤謬の例

## 前後関係と因果関係の混同（前後即因果の誤謬）

- 説明
    - A が起きてから B が起きたという事実を捉えて、A が B の原因であると早合点すること
- 誤謬の例
    - あなたがこのアパートに引っ越してくるまで暖房設備には何の問題もなかった
    - 高校や大学への進学率はかつてないほど高くなってきた。それでも、少年犯罪や問題を抱えた若者はかつてより多い。よって、若者が教育によって堕落させられている。
- 筆者的見解
    - 「因果関係」と「前後関係」と「相関関係」は違う
    - 前後関係はあるように見えるが、因果(原因と結果)は伴っていない

<br>

## 相関関係と因果関係の混同（擬似相関）

- 説明
    - 相関関係があるものを短絡的に因果関係があるものとして扱う
- 誤謬の例
    - 撲滅された病気の数とテレビの普及には相関関係がある。よってテレビが普及すれば病気が撲滅される。
    - アイスクリームの売り上げが最も高い時期には、プールでの溺死事故も最も多い。アイスクリームの売り上げ増が溺死増の原因。
- 筆者的見解
    - 「因果関係」と「前後関係」と「相関関係」は違う
    - 相関しているように見えるが、因果(原因と結果)は伴ってない

<br>

## 因果関係の逆転

- 説明
    - 因果関係を逆転させて主張する
- 誤謬の例
    - 火災現場に出動する消防士が多いほど火災の規模は大きい。従って、出動する消防士が多くなることが、火災が大きくなる原因だ。
    - 車椅子は危険である。なぜなら、車椅子に乗っている人は事故に遭ったことがあるから。
    - バスケットボールの選手は身長が高い。よってバスケットボールをすると背が伸びる。
- 筆者的見解
    - 「因果関係」と「前後関係」と「相関関係」は違う
    - 相関しているように見えるが、因果(原因と結果)が逆転している

## テキサスの狙撃兵の誤謬（クラスター錯覚）

- 説明
    - 本来相関のないものを相関があるとして扱う。
- 誤謬の例
    - 上官が狙撃兵に腕前を問うたところ、遠くにある壁の標的の真中に命中しているのを指し示したため腕前に感心したが、実際は壁の銃痕にあとから標的を描いただけだった。
    - 電力線が健康上の悪影響を引き起すかどうか調査したところ、近いところに住む人々の小児白血病の発生率は4倍だったが、実際は多重比較問題によって統計的に有意差が出ただけだった。
- 筆者的見解
    - 「因果関係」と「前後関係」と「相関関係」は違う
    - 相関しているように見えるが、そもそも相関関係などない

<br>

## 滑り坂論法（ドミノ理論）

- 説明
    - 可能性の低い因果関係を無理矢理繋げて、何らかの事物の危険性を主張すること
- 誤謬の例
    - 風が吹けば桶屋が儲かる
    - バタフライ効果
- 筆者的見解
    - 「因果関係」と「前後関係」と「相関関係」は違う
    - 因果はあるのかもしれないが、そもそも可能性が低いし再現性もない

## 公正世界誤謬

- 説明
    - 人間の行いに対して公正な結果が返ってくるものである、と考える認知バイアス
- 誤謬の例
    - 我欲に天罰が下った。
    - ハンセン病に罹患するのは宿業を負ったものが輪廻転生したから。
    - カーストが低いのは前世でカルマが悪かったから。
- 筆者的見解
    - 思い込み。認知バイアス。

## 例外の撲滅

- 説明
    - 例外を無視した一般化を元に論旨を展開すること
- 誤謬の例
    - ナイフで人に傷をつけるのは犯罪。外科医はナイフで人に傷をつける。従って、外科医は犯罪者。（ただの手術）
    - 雨の日には傘を差す。お姉さんが傘を差している。従って、今日は雨が降っている。（ただの日傘）
- 筆者的見解
    - 思い込み。認知バイアス。

## 誤った二分法（白黒思考）

- 説明
    - 選択肢をいくつか提示し、それ以外に選択肢がないという前提で議論を進めること
- 誤謬の例
    - このまま借金取りに悩まされる人生を送るか、自殺するか
    - あなたがたは、解決策の一部であるか、さもなくば問題の一部だ
- 筆者的見解
    - 思い込み。認知バイアス。

## 間違った類推

- 説明
    - 重大な相違を無視して事象の類似性に基づいて論証（類推）すること
- 誤謬の例
    - 酒とコーヒーは似たような嗜好品。飲酒は法律で規制されている。よって、コーヒーを飲むのは法律で規制されるはず。
- 筆者的見解
    - 似ているだけで同じではないしそもそも別物なので類推もできない
    - そもそも類推という論証自体が脆弱

<br>

## 多義語の誤謬

- 説明
    - 複数の意味をもつ語を使って三段論法を組み立てること
- 誤謬の例
    - 車（自動車）の運転には免許が必要だ。自転車は車（車両）である。したがって自転車の運転には免許が必要。
- 筆者的見解
    - 似ているだけで同じではないしそもそも別物なので類推もできない
    - 適切な単語を用いて論証しましょう

<br>

## 論点先取

- 説明
    - 証明すべきはずの命題が仮定や前提に含まれてしまっている、という状態
- 誤謬の例
    - かわいくない猫は存在しないと仮定する。猫は日本に生息している。したがって、日本の猫はかわいい。（仮定の時点で「猫はかわいい」という意味が含まれている）
    - ポールは嘘を言っていない。ポールは何かを話している。したがって、ポールは本当の事を言っている。（仮定の時点で「本当の事を言っている」という意味が含まれている）
- 筆者的見解
    - 形式的には間違ってないけど、わざわざ論理的に展開する内容ではない
    - 循環論法

<br>

## 曖昧語法

- 説明
    - 曖昧な文形で主張をすること
- 誤謬の例
    - 十代の若者に自動車を運転させるべきではない。それを許すのは非常に危険。（若者が危険な目に遭うから？若者な危険な行為をするから？）
- 筆者的見解
    - 「なぜなら」を述べるべき

<br>

## 連続性の虚偽

- 説明
    - 述語の曖昧性により常識的な認識とのズレが生じる誤謬
- 誤謬の例
    - 砂山のパラドックス（砂山から砂粒を取り去っても依然として砂山のままだが、それから何度も取り続けて、最終的に一粒だけが残ったとき、その一粒だけを指して「これは砂山だ」と言えるか）
    - テセウスの船（ある物体において、それを構成するパーツが全て置き換えられたとき、過去のそれと現在のそれは「同じそれ」だと言えるか）
- 筆者的見解
    - 定義や境界値が明確でなく曖昧なまま命題を扱っている

<br>

## 多重質問の誤謬

- 説明
    - 被質問者が異議を申し立てずに答えた場合、ある告発を認めたことになるような前提を含んだ質問形式
    - 質問の前提に証明されていない事柄が含まれており、「はい」と答えても「いいえ」と答えてもその前提を認めたことになるという質問形式
- 誤謬の例
    - 君はまだ天動説を信じてる？
    - お前はまだ奥さんを虐待しているのか？
- 筆者的見解
    - 「そもそもその質問はなんなの？」という異議申し立てをしましょう

<br>

## おべっか

- 説明
    - 2つの質問を同時にするもので、1つは被質問者が「はい」と答えたくなるもので、もう1つは質問者が「はい」と答えてほしいものである。
- 誤謬の例
    - あなたはいい人で私に5ドル貸してくれますか？（はい→5ドル貸さないといけない。いいえ→いい人ではなくなる）
- 筆者的見解
    - 「AについてはYesで、BについてはNo」という分割をしましょう
